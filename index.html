Top projects

multi-agents Reinforcement learning (MARL)

With initial guidance from Dr. Farhad Fazileh

In this project, I explore agent interactions within mean‑field game theory which include a large number of agents. The AI algorithms are based on reinforcement learning, presenting substantial complexities across various game scenarios. This project is still ongoing.

Bachlor's thesis: machine learning in wearable sensors

Supervisor: Dr. Peyman sahebsara

In this project, I explored wearable technologies and various types of sensors, with a special focus on fiber optic sensors, which have emerged as a powerful force in the wearable tech field. I then outlined the fundamental concepts and steps for implementing machine learning in wearable sensors, reviewed existing research in this area, and proposed solutions to address the challenges encountered.
